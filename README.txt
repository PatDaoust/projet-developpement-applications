Ce produit est une application Winforms, écrit en C#. 

Dans sont êtas présent de development , il est ouvrable avec Visual Studio 2019 et Visual Studio 2022. 
Utiliser une de ces versions de Visual Studio pour lancer le fichier central, qui ce trouve dans a 
projet-developpement-applications\Scrum 1\gestionRH\gestionRH.sln
Dans Visual Studio, cliquer la flèche verte « Start » pour lancer l’application
Les employés administratif peuvent choisir de saisir leur propre feuille de temps ou de valider les feuilles de temps d'autres employés.
Les autres employés ne peuvent entrer que leur propre feuille de temps.
